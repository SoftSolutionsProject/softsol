Ementa do curso

Módulo 1: Introdução ao Python

  • História e evolução do Python
  • Instalação e configuração do ambiente de desenvolvimento
  • Estrutura básica de um programa Python
  • Tipos de dados e variáveis

Módulo 2: Estruturas de Controle

  • Operadores e expressões
  • Estruturas condicionais (if, else, elif)
  • Loops (for, while)
  • Compreensões de listas e expressões geradoras

Módulo 3: Funções e Módulos

  • Definição e uso de funções
  • Argumentos e parâmetros
  • Importação de módulos e bibliotecas
  • Introdução à programação orientada a objetos

Módulo 4: Manipulação de Dados

  • Listas, tuplas e dicionários
  • Manipulação de arquivos (leitura e escrita)
  • Uso de bibliotecas para manipulação de dados (pandas, numpy)
  • Introdução ao tratamento de exceções

Módulo 5: Desenvolvimento de Aplicações

  • Automatização de tarefas com scripts Python
  • Desenvolvimento de aplicações web com Flask
  • Introdução ao Django
  • Integração com bancos de dados

Módulo 6: Projetos Práticos

  • Desenvolvimento de projetos Python completos
  • Trabalho em equipe e colaboração
  • Apresentação de portfólio e melhores práticas para busca de emprego
